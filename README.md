# Tiny Universe

## Overview
A wearable electronics project that translates body movement and environmental changes into light and sound.

## Features
- Light sensor triggers LEDs when hand closes or in darkness
- Motion activates sound feedback
- Wearable circuit integrated into clothing

## Materials
- Circuit Playground
- Light sensor
- DIY pressure sensor
- Translucent plastic sheets
- Metal rings (as decoration and wiring)

## Concept
This project explores how wearable technology can reveal hidden body signals through interactive feedback.
